# Tus Básicos 🛠️

Una colección de herramientas web esenciales diseñadas para el uso diario. Este proyecto es una práctica de fundamentos de desarrollo web, enfocada en la manipulación del DOM y la lógica de programación.

## Funcionalidades
- **Calculadora de Porcentajes:** Interfaz limpia para calcular montos y descuentos rápidamente.
- **Gestor de Tareas (To-Do List):** Permite agregar y visualizar tareas pendientes.
- **Navegación por Pestañas:** Transición fluida entre herramientas sin recargar la página (SPA feel).
- **Modo Oscuro:** (En desarrollo) Opción para cambiar el tema visual.

## Tecnologías
- **HTML5:** Estructura semántica.
- **CSS3:** Flexbox, variables CSS y diseño responsivo.
- **JavaScript (Vanilla):** Lógica de interacción, eventos y manipulación del DOM.
